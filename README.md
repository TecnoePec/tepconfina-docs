# TepConfina Docs

Documentacao tecnica do TepConfina - Sistema de Gestao de Confinamento Bovino.

**Site:** https://tecnoepec.github.io/tepconfina-docs/

## Desenvolvimento Local

```bash
pip install -r requirements.txt
mkdocs serve
```

Acesse http://localhost:8000

## Deploy

O deploy e automatico via GitHub Actions ao fazer push na branch `main`.
